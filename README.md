# HD BUSINESS ERP V1.0

一个带有基于角色访问控制 (RBAC) 的企业资源规划 (ERP) 应用程序。

## 功能特点
- **身份验证**：企业登录系统。
- **基于角色访问控制 (RBAC)**：支持管理员和操作用户角色。
  - *管理员*：完全访问权限，用户管理，可删除发票。
  - *用户*：操作访问权限，权限受限。
- **用户管理**：添加、编辑和查看用户（仅限管理员）。
- **角色切换**：在导航栏中轻松切换角色以进行测试。
